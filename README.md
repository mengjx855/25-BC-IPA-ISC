## A microbiota-IPA axis facilitates intestinal stem cell-mediated regeneration in colitis through a Hopx-associated program

**Folder and File Structure**
- pipeline: metagenomics and genomics pipeline, including abundance estimation of key species, identification of key genes, etc.
- scripts: some R/Shell function used in this program
- data: the intermediate data during analysis
- FigureX: the scripts and necessary files for each figure

**The database and software used in this study as follow:**
- fastp (v0.24)
- Bowtie2 (v2.5.4)
- Minimap2 (v2.28-r1209)
